# CLI

command line interface 명령어 정리

#### pwd

현재 위치한 폴더를 출력하는 명령어

```shell
pwd
```



#### ls

list의 약자로 현재폴더에 있는 모든 파일과 폴더를 출력하는 명령어

```shell
ls # 숨김폴더는 출력안함
ls -a # 숨김폴더, 파일까지 모두 출력함
```



#### cd

change directory의 약자로 폴더를 이동하는 명령어

```shell
cd <PATH> # 이동하고 싶은 폴더를 PATH에 입력
```



#### 생성

- `touch`: 파일을 생성
- `mkdir`: 폴더를 생성
